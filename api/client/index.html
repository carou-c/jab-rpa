JabRpaClient

Low-level gRPC client for the JAB server.

Wraps each RPC method defined in the jab.proto service definition. Not intended for direct use — accessed through JabDriver.

accessible_click 

accessible_click(element: Element) -> None

Perform an accessible action click via the JAB API.

Parameters:
  • element (Element) –

    The jab.Element to click.

  • action (Action) –

    The jab.Action to perform.
