JabRpaClient

Low-level gRPC client for the JAB server.

Wraps each RPC method defined in the jab.proto service definition. Not intended for direct use — accessed through JabDriver.

do_action 

do_action(
    action: Action,
    selector: str,
    timeout_ms: int | None,
    refresh_before_fail: bool,
) -> None

Perform an accessible action on the first element matching a selector.

Parameters:
  • action (Action) –

    The Action to perform.

  • selector (str) –

    CSS-like selector string.

  • timeout_ms (int | None) –

    None for default (60s), 0 for no wait, or a positive integer for max milliseconds.

  • refresh_before_fail (bool) –

    If true, refresh the tree after timeout before the final check.
Raises:
  • JabInvalidArgumentError –

    The selector string is malformed.

  • JabNoWindowError –

    No window has been selected yet.

  • JabTimeoutError –

    The element did not appear within the timeout.

  • JabInternalError –

    The action could not be performed.
