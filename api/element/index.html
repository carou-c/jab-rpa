Element

Wraps a JAB accessible element with readable properties and interaction methods.

Provides direct access to all element attributes exposed by the Java Access Bridge, plus convenience methods for clicking and typing.

Instances are created by Locator.matching(), Locator.first_matching(), Locator.wait_for(), Element.children(), and Element.parent().

accessible_action property 

accessible_action: bool

Whether the element supports accessible actions (e.g. clicking via JAB).

accessible_selection property 

accessible_selection: bool

Whether the element supports accessible selection operations.

accessible_text property 

accessible_text: bool

Whether the element supports accessible text operations.

children_count property 

children_count: int

Number of direct children in the accessibility tree.

depth property 

depth: int

Depth of element within the context tree.

description property 

description: str

Accessible description.

height property 

height: int

Height of the element's bounding box in pixels.

index_in_parent property 

index_in_parent: int

Index of this element within its parent's children list.

name property 

name: str

Accessible name of the element.

role property 

role: str

Accessible role (e.g. "push button", "checkbox").

states property 

states: list[str]

Current states as localized strings (e.g. ["enabled", "focusable"]).

states_en_us property 

states_en_us: list[str]

Current states as en-US strings (e.g. ["enabled", "focusable"]).

width property 

width: int

Width of the element's bounding box in pixels.

x property 

x: int

Screen X coordinate of the element's bounding box origin.

y property 

y: int

Screen Y coordinate of the element's bounding box origin.

accessible_click 

accessible_click() -> None

Click using the JAB accessibility API (not pyautogui). This does not move the mouse.

click 

click(clicks: int = 1, interval: int | float = 0.0) -> None

Click at the element's center using pyautogui.

Moves the mouse to the center of the element's bounding box and performs a pyautogui click.

Parameters:
  • clicks (int, default: 1 ) –

    Number of clicks (default 1).

  • interval (int | float, default: 0.0 ) –

    Seconds between clicks (default 0).

click_and_type 

click_and_type(
    text: str,
    clicks: int = 1,
    interval_text: int | float = 0.0,
    interval_clicks: int | float = 0.0,
) -> None

Click then type text into the element.

Moves the mouse to the element's center, clicks, then types the given text using pyautogui.

Parameters:
  • text (str) –

    Text to type after clicking.

  • clicks (int, default: 1 ) –

    Number of clicks before typing.

  • interval_text (int | float, default: 0.0 ) –

    Seconds between keystrokes.

  • interval_clicks (int | float, default: 0.0 ) –

    Seconds between clicks.
